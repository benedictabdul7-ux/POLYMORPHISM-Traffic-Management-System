# POLYMORPHISM-Traffic-Management-System
A lightweight Python project demonstrating key Object-Oriented Programming (OOP) concepts, specifically Inheritance and Polymorphism.

## 📌 Description
This project simulates a set of smart traffic control devices—including traffic lights, speed cameras, pedestrian signals, and emergency sirens. Built on a shared `TrafficDevice` parent class, each device subclass overrides the `activate()` method to execute its own unique action, showing how Python handles dynamic method invocation.

## 🧬 Key Features
* **Base Class (`TrafficDevice`):** Establishes a unified interface for all traffic system components[cite: 1].
* **Polymorphism in Action:** Subclasses (`Trafficlight`, `SpeedCamera`, `PedestrianSignal`, `EmergencySiren`) implement specialized responses when triggered[cite: 1].
* **Clean Code Design:** Demonstrates iterating through a list of diverse objects without relying on complex `if/else` logic[cite: 1].
